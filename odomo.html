Pluviometrie project

Creation and programming of a small personal weather station

Technologies

  • Java

  • NetBeans

Requirement Specification

  • Use of Java language to manipulate weather and heating management
  • Respect the structure of the code provided, in particular the different classes and their attributes.
  • Program source code, following coding conventions and appropriate documentation.
  • Design document detailing algorithmic choices and personal contributions to the project.

Competences

  • Mastery of the basic concepts of Java programming
  • Ability to write effective unit tests to validate the proper functioning of the different parts of this project.
  • Ability to design a friendly and intuitive user interface to facilitate interaction with the user.
  • Skill in identifying, analyzing and resolving problems encountered throughout the development in this project.
